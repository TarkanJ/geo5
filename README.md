# GEO5
Survey/Geodetic application

## Spuštění programu
* lze spustit úpravou skriptu "geo5-spust.sh"

* nebo:

Linux verze
-----------
gcc geo5.c -o geo5
echo "Spoustim GEO5"
./geo5

Windows verze
-------------
gcc geo5_revival.c -o geo5 -lm
geo5.exe
